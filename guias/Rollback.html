Guía para hacer Rollback

Objetivo

Reestablecer la versión estable anterior de un nivel de control de configuración

Rollback en git

Pasos para hacer rollback si se está usando git

  • Usa esta referencia para regresar a un commit anterior
    • Si estás en el nivel estático, regresa al commit de la release más reciente
    • De lo contrario, regresa al último commit
  • Si estás en el nivel estático o en la rama test, seguir los mismos pasos del proceso de deploy externo
