Guía de Actualizacion de Plan de Proyecto

V1.1

Objetivo

Actualizar el plan de proyecto para conocer el estatus actual del proyecto y  poder determinar futuras estrategias definiendo el costo y esfuerzo.

Plan de Riesgos

 De encontrarse Riesgos, se deberá de actualizar el plan de riesgos, para esto seguir el proceso de riesgos.

Link al proceso de riesgos.

Estimaciones

Para actualizar el plan de proyecto y obtener las nuevas estimaciones es necesario ingresar a la carpeta de plan de proyecto Impulse.

Liga: https://docs.google.com/spreadsheets/d/1v0z_ykBQmvAHtRDiLpDriwPO6rLBN9NVuWaM-To1Ztk/edit#gid=2063131004

En la carpeta se seleccionará la pestaña correspondiente a cada proyecto bajo el nombre Plan de Proyecto TuProyecto

  • Si quedó trabajo pendiente de la iteración, este se agrega en puntos en la fecha que corresponda en la fila de Trabajo Nuevo. Este solito actualizará el plan de proyecto y ajustará las fechas.
