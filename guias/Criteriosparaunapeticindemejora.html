Criterios para una petición de mejora

Objetivo

        El objetivo de este documento es informar acerca de las propuestas de mejora a procesos (PIPs) así como establecer los criterios para que una petición sea válida.

Definición

Propuesta de Mejora de Procesos

Una propuesta de mejora aporta al proceso de mejora continua del modelo de calidad CMMI, de tal manera que beneficie a la organización. Esta propuesta no tiene un tiempo definido, es decir, puede ser propuesta en cualquier momento y tener una duración indefinida.

Criterios

Para que una propuesta sea considerada una propuesta de mejora de procesos debe de estar dentro de al menos una de las siguientes categorías.

  1. Aumento de flujo de valor
  2. Ciclos de retroalimentación más cortos
  3. Aprendizaje continuo y experimentación.

Tiempo

Las propuestas de mejora a procesos, no tienen definido un tiempo de aplicación ni de ejecución, es decir, estas pueden surgir en cualquier fase de un proyecto o en cualquier momento dentro de la organización.
