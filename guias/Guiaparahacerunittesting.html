¿Cómo hacer tests?

Para realizar nuestros tests, vamos a utilizar la librería que proporciona django llamada test.

Dentro del app donde se encuentra la implementación de tu US, crea un archivo con el nombre “tests.py”

En este archivo, se crearán los tests.

Es muy importante que todos tus tests comiencen con la palabra “test” y que dichos tests reciba como parámetro “TestCase”

Ej:

Para correr tus tests, corre la siguiente línea de comando en la terminal:

$python3 manage.py test app_name
