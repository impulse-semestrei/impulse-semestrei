Guía para hacer rollback

Objetivo

        Reestablecer la versión estable anterior de un nivel de control de configuración

Rollback en git

Pasos para hacer rollback si está usando git

  • Use esta referencia para regresar a un commit anterior
  • Si está en el nivel estático, regrese al commit de la release más reciente
  • De lo contrario, regrese al último commit
  • Si está en el nivel estático o en la rama test, siga los mismos pasos del proceso de deploy externo
