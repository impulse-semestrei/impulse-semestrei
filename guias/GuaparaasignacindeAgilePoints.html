Guía para asignación de Agile Points

Objetivo

Estimar el tamaño de cada User Story.

Recursos

  • Pizarrón y plumones

  • Post-its con las User Stories

Actividades sugeridas

  1. Crear una tabla en la que los encabezados sean los primeros seis números de la serie de Fibonacci. (Ejemplo en imagen al final del documento)
  2. Repartir la misma cantidad de User Stories entre todos los miembros del equipo.
  3. Cada miembro acomoda en la tabla las User Stories, en la columna que considere adecuada.
  4. Para cada User Story ya ingresada en la tabla:
  1. Se compara su posición con respecto a las demás User Stories.
  2. Se piden objeciones en cuanto a su posición en la tabla. Si no hay, se procede a analizar la siguiente User Story.
  3. En caso de haber objeciones, se discute y se cambia de posición.
  4. Se regresa al paso `i` para la misma User Story.
