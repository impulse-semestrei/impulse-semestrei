Realizar un WBS

Objetivo

Una WBS (Work Breakdown Structure) de alto nivel puede servir para estructurar la estimación inicial. El desarrollo de una WBS divide el proyecto global en un conjunto interconectado de componentes manejables.

Acciones Recomendadas

  • Tener un proyecto con objetivos. El nombre del proyecto irá en la cabeza del árbol del WBS.
  • Definir las acciones necesarias para cumplir los objetivos a nivel general.
  • Las acciones definidas se deben descomponer en tareas, procura granular lo más que se pueda.
  • Una vez definidas esas tareas, se le debe asignar el tiempo que se estima realizar para cada tarea.
